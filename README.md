 <a href="https://java-project-72-a7uc.onrender.com">Page-analyzer</a>

Page Analyzer is a site that analyzes specified pages for SEO suitability.
You receive information about the site title, its description, status code and p1. 

To use the site, you need to enter the site address into the form like "https://example.com/" and click the "check" button.

After adding a site to the check list, click on its link and start the check by button.
